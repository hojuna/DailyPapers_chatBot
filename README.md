## 프로젝트 개요
이 프로젝트는 Hugging Face의 [논문 페이지](https://huggingface.co/papers)에서 논문들의 요약을 크롤링하고, 한국어로 번역하여 PDF로 저장하는 프로그램입니다. 

## 주요 기능
- Hugging Face 논문 페이지에서 모든 논문 요약 크롤링
- 요약을 한국어로 번역 (Papago)
- 번역된 요약을 하나의 PDF 파일로 저장
